Informações sobre o Aplicativo Shiny

Público-Alvo

O público-alvo do aplicativo inclui:

  • Pesquisadores e acadêmicos: Para criar nuvens de palavras a partir de seus textos de pesquisa e visualizar padrões de frequência de palavras.
  • Professores e alunos: Para fins educacionais, como análise de textos e aprendizado sobre análise de texto e visualização de dados.
  • Profissionais de marketing: Para analisar feedback de clientes e identificar palavras-chave importantes em análises de sentimento.
  • Escritores e blogueiros: Para visualizar palavras recorrentes em seus textos e melhorar a escolha de palavras.
  • Qualquer pessoa interessada em análise de texto: Que queira explorar a frequência e a relevância das palavras em textos.

Tema Abordado e Objetivos

Tema: Análise de texto e visualização de dados.

Objetivos:

  • Permitir aos usuários criar nuvens de palavras a partir de textos digitados ou carregados.
  • Oferecer opções para personalizar a nuvem de palavras, como selecionar o idioma, remover palavras específicas e ajustar a quantidade de palavras.
  • Permitir o download da nuvem de palavras em formato HTML.
  • Exibir uma tabela de frequência das palavras utilizadas.
  • Fornecer o código-fonte do aplicativo para aprendizado e modificação.

Pré-Requisitos

Quais são os conhecimentos necessários para a compreensão dos conceitos abordados nesse aplicativo?

  • Conhecimento básico de R e Shiny: Para entender como o aplicativo é construído e como as interações funcionam.
  • Conceitos de análise de texto: Como limpeza de texto, remoção de palavras comuns (stopwords) e a criação de nuvens de palavras.
  • Conhecimento básico sobre visualização de dados: Entendimento do propósito de visualizações como nuvens de palavras.
  • Familiaridade com a interface do usuário do Shiny: Para usar e interagir com o aplicativo eficientemente.

Instruções para a Utilização do Recurso

Instruções para a utilização do recurso:

  1. Selecione o Idioma: Escolha o idioma do texto para que o aplicativo utilize as stopwords corretas (palavras comuns que devem ser removidas).
  2. Escolha a Fonte do Texto:
    • Digite seu Texto: Insira o texto diretamente na caixa de texto fornecida.
    • Carregue um Arquivo: Selecione um arquivo de texto (.txt ou .csv) contendo o texto.
  3. Remover Palavras Específicas (opcional): Se desejar remover palavras específicas da nuvem, ative a opção e insira as palavras.
  4. Ajuste a Quantidade de Palavras: Use o controle deslizante para definir o número de palavras a serem exibidas na nuvem.
  5. Escolha a Fonte e a Cor do Fundo: Selecione a fonte para a nuvem de palavras e a cor do fundo desejada.
  6. Escolha o formato da Nuvem: Selecione o formato da nuvem entre os disponíveis.
  7. Visualize a Nuvem: A nuvem de palavras será gerada e exibida na área principal.
  8. Faça o Download: Use o botão de download para salvar a nuvem de palavras como arquivo HTML.
  9. Veja o Código Fonte: Navegue até a aba de Código Fonte para visualizar o código do aplicativo.

Exemplo de Utilização

Exemplo de utilização:

Você tem um arquivo de texto (.txt ou .csv) e deseja criar uma nuvem de palavras para uma análise.

    Passos:
    • Você seleciona o idioma “Português” no aplicativo.
    • Carrega um arquivo .txt contendo o texto.
    • Você opta por remover palavras específicas como nomes próprios e termos irrelevantes.
    • Então ajusta a quantidade de palavras para 50 e escolhe a fonte “Times New Roman” com fundo branco e paletas de cores escuras.
    • Faça os ajustes desejáveis de cores e formato.
    • A nuvem é modificada a cada passo, ou alteração feita.
    • Usa-se o botão de download para salvar a imagem da nuvem de palavras.
    • Você também pode navegar para a aba de Código Fonte para revisar e possivelmente ajustar o código conforme necessário para outros projetos.
